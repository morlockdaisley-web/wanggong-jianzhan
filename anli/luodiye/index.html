// 02 课程体系

四阶进阶,从拖积木到打竞赛

// L1

图形化启蒙

适合 7-9 岁 · 零基础

用 Scratch 积木拖出会动的角色和能玩的小游戏。先建立「我能做出来」的信心,再谈其他。

// L2

Python 入门

适合 9-11 岁

从积木换成真代码。变量、循环、条件不靠背,写文字冒险游戏和查天气的小程序,用会为止。

// L3

算法思维

适合 10-12 岁

排序、枚举、递归,配 CSP-J 入门题单。重点练一件事:把一道大题拆成能下手的小步骤。

// L4

竞赛方向

适合 12 岁以上

面向 CSP-J/S 与信奥路线的小班,按孩子的实际水平定目标、定节奏,不搞大锅饭式刷题。
